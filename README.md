Capyfetch is a simple fetch that displays system information with a capybara.

# Installing
## Arch
Capyfetch is available in AUR
```bash
yay -S capyfetch
```
or
```bash
paru -S capyfetch
```
## Other distros
Request: base-devel, git
```bash
git clone https://github.com/Senya44ka/capyfetch/ && cd capyfetch && makepkg -si
```
